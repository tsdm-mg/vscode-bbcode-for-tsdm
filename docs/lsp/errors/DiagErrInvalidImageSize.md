# DiagErrInvalidImageSize

`DiagErrInvalidImageSize`是值在`[img]`标签中标注的图片大小的写法有问题。

## Fix

遇到这个写法，请调整成正确的`width,height`也就是`宽,高`格式。

## Example

```console
# Bad
[img=1920][/img]
[img=1080][/img]

# Good
## 填写图片尺寸
[img=1920,1080][/img]
## 填写宽度，让高度自动调整
[img=1920,0][/img]
## 填写高度，让宽度自动调整
[img=0,1080][/img] # 宽度为0表示自动调整
```

## Note

**此处记载的`[img]`用法以tsdm为准，其它论坛或用途中的效果和用法可能不同。**

- 填写的图片宽高可以不符合图片的实际宽高。
- 在填写图片宽和高时其中一条边可以写0，表示这条边的数值按照图片大小和网页布局自动调整。
- 由于网页布局有最大宽度，一般推荐直接填写图片的原尺寸或者只填写宽度，让高度自动调整。
- 在网页布局中，上下高度数值大的图片（通常说的长图）推荐填写一个最大高度，让宽度自动调整。
